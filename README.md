# Calculator

A project built as part of [The Odin Project](https://www.theodinproject.com/) (Foundations) — the capstone project of the Foundations path.

## Description
An on-screen calculator built with vanilla JavaScript, HTML, and CSS. 
Supports basic arithmetic operations with edge case handling.

## Features
- Addition, subtraction, multiplication, division
- Chained calculations (result of first operation becomes first number of next)
- Clear button to reset the calculator
- Error handling for division by zero
- Rounded display for long decimals

## Technologies
- HTML
- CSS
- JavaScript (DOM manipulation)
